# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка Telegram бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен

## 3. Создание .env файла

Создайте файл `.env` в корне проекта:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

**Примечание:** `TELEGRAM_CHAT_ID` больше не нужен! Бот автоматически отправляет товары всем пользователям, которые подписались через команду `/start`.

## 5. Тестирование

Сначала протестируйте парсер:

```bash
python test_parser.py
```

Если товары не находятся, возможно нужно использовать Selenium. Добавьте в `.env`:

```
USE_SELENIUM=true
```

И установите ChromeDriver: https://chromedriver.chromium.org/

## 6. Запуск бота

```bash
python main.py
```

Бот начнет парсить товары и отправлять их всем подписанным пользователям!

## 7. Использование бота

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start` чтобы подписаться на рассылку
3. Бот будет автоматически отправлять вам новые товары
4. Используйте `/stop` чтобы отписаться от рассылки
5. Используйте `/status` чтобы проверить статус подписки

## Настройка интервала

В файле `config.py` можно изменить:
- `PARSING_INTERVAL` - как часто парсить (в секундах)
- `MAX_PRODUCTS_PER_MESSAGE` - сколько товаров отправлять за раз
